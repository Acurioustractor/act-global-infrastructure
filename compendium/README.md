# ACT Infrastructure Compendium

This folder contains the project-specific compendium for ACT Global Infrastructure — aligning the operational backbone to ACT's master philosophy.

## Structure

```
compendium/
├── README.md              ← You are here
├── identity.md            ← Project identity, philosophy alignment, LCAA
├── stories/
│   └── index.md           ← Impact pathways (how infrastructure enables projects)
├── operations/
│   ├── authority.md       ← Authority check, consent, handover
│   └── partners.md        ← Ecosystem connections
└── roadmap.md             ← 2026 focus and current status
```

## How to Use

- **New contributor?** Start with `identity.md`
- **Understanding the system?** See `stories/index.md` for how infrastructure connects to community impact
- **Architecture questions?** Check the wiki at `apps/command-center/public/wiki/`
