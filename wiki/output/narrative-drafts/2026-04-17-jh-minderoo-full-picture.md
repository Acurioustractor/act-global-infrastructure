## The Brave Ones — Where It Fits

The Brave Ones is not a separate program. It is the **human spine** of The Three Circles.

```
THE THREE CIRCLES                    THE BRAVE ONES
─────────────────                    ──────────────

Circle 1 (Centre/Platform)    ←──   Journal data feeds evidence base
                                     Stories linked to program data + Hansard

Circle 2 (The Ten)            ←──   The 7 pilot communities ARE anchor communities
                                     Journals + rewards + exchanges = the daily practice

Circle 3 (Artifact)           ←──   "The Brave Ones" book series + exhibitions = Year 3 artifact
                                     Young people's own work, not written about them
```

## The Brave Ones — Three Layers

### Layer 1: The Journal (daily practice)

**CEO Review Decision: Approach C — Guided Access + Existing Apps**

No custom app in Year 1. iPads locked via Apple's built-in Guided Access to Procreate (drawing) + Voice Memos (audio). Workers batch-upload to Empathy Ledger weekly. $80K saved, ships in a week. Evaluate custom app in Year 2.

```
┌──────────────────────────┐     ┌──────────────────────────┐
│    iPAD + APPLE PENCIL   │     │    PHYSICAL A5 BOOK      │
│    (Guided Access)       │     │                          │
│                          │     │  - Stickers, cut-outs    │
│  PROCREATE ($15/device)  │     │  - Family can write in   │
│  - Draw / finger paint   │     │  - Workers can write in  │
│  - Digital collage       │     │  - Printed pages from    │
│  - Professional tools    │     │    the iPad stuck in     │
│                          │     │  - Stays with the kid    │
│  VOICE MEMOS (free)      │     │                          │
│  - Audio recording       │     │  "Can I grab my journal?"│
│  - Better than writing   │     │   ← the goal state       │
│    for low literacy      │     │                          │
│                          │     │                          │
│  CLOSED SYSTEM           │     │                          │
│  (Guided Access = no     │     │                          │
│   browser, no YouTube)   │     │                          │
│                          │     │                          │
│  Workers upload to EL    │     │                          │
│  weekly (consent gated)  │     │                          │
└──────────────────────────┘     └──────────────────────────┘
```

**Consent workflow (Approach C):**
- Workers use Empathy Ledger's existing consent envelope when uploading
- Each piece tagged: share / don't share / share anonymously
- Guardian consent on paper intake form, recorded in EL
- Verbal consent recorded via Voice Memo before each upload session
- Cultural protocols through each community's existing authority structure

**From the Oonchiumpa conversation:**
- Girls sit and paint for 2 hours unprompted
- Hudson sat on a computer for 2 hours — they engage with technology
- Audio recording + transcription > asking kids to write (literacy barriers)
- A closed iPad system where they can ONLY do the journal — "they would dig that"
- The physical object matters — personal, tactile, theirs

## Pilot: Oonchiumpa, June 2026

**Proceeds on existing ACT/Oonchiumpa budget — independent of Minderoo confirmation.**
This is a proof point, not a promise.

```
2026-06-01  iPads + Guided Access + physical journals set up
2026-06-08  Brisbane inspiration trip = first domestic exchange
2026-08-01  First EL data from Oonchiumpa journals
2026-09-01  Pilot review — what's working, what's not
2026-12-01  First inter-community exchange (Alice Springs <> Mount Isa or Townsville)
```

**Validation criteria:**
- [ ] At least 3 kids voluntarily journal in the first month
- [ ] At least 1 kid asks for their journal unprompted (the "goal state")
- [ ] Worker successfully uploads to EL at least once per week
- [ ] Physical journals show use (not blank after 4 weeks)
- [ ] Brisbane trip produces at least 1 shareable exchange moment
- [ ] Kids prefer drawing/audio over typing (validates Approach C)
- [ ] Guided Access stays locked (kids don't bypass it)

## Evaluation Framework (CEO Review — Outside Voice Finding)

Wired into Oonchiumpa's existing ANU True Justice research partnership:

| Window | What's measured |
|--------|----------------|
| Intake (baseline) | Engagement, justice status, self-reported wellbeing |
| 3 months | Journal engagement frequency, self-initiated journaling |
| 6 months | School/training re-engagement, exchange participation |
| 12 months | Re-offending rates, alumni return rate, peer connections maintained |

**Success = artifacts + engagement + outcomes:**
- Books/exhibitions prove the model produces output
- EL data proves young people are choosing to participate
- 12-month follow-up proves the model affects trajectories
- Alumni data proves the model is self-sustaining
