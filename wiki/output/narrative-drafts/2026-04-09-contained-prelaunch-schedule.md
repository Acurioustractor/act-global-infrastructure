## Daily 5-minute rhythm

Each day follows the same 5-minute loop:

```bash
# 1. Check the strategy review for any drift
cat wiki/narrative/contained/STRATEGY-REVIEW.md | head -25

# 2. Open the day's draft (already in wiki/output/narrative-drafts/)
# Edit in Obsidian or VS Code

# 3. After publishing, log the deployment
node scripts/narrative-log-deployment.mjs <claim-id> <channel> \
  --source <published-url> \
  --variant "<the actual line you used>"

# 4. (Auto-runs after #3) — strategy review re-generates
```

The system updates itself between days. If a post lands and the engagement is strong, the next day's draft can pivot to amplify the claim that worked. If it lands flat, the next day's draft picks the next gap.
