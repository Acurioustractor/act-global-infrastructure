## How to use this

1. Before drafting any post, open this file and the [STRATEGY-REVIEW.md](STRATEGY-REVIEW.md) (once it exists).
2. Find under-deployed claims — draft against those, not the ones we've already run.
3. Log deployments with `node scripts/narrative-log-deployment.mjs <claim-id> <channel>` so the strategy review stays accurate.
4. Claim file naming: `claim-<short-hook>.md`. Keep slugs short and specific.

## Campaign frames

The art campaign uses these frames in rotation so the tone doesn't flatten into outrage or technocracy:

| Frame | What it does | Example claim |
|---|---|---|
| structural | Names a system-level pattern | information-asymmetry-is-the-system |
| confrontational | Forces a binary or moral choice | keyhole-or-window |
| evidentiary | Lets the numbers carry the weight | 58x-return-on-donation |
| testimonial | First-person voice from a storyteller | laquisha-oppression |
| invitational | Offers a way in (subscription, collaboration) | civicgraph-for-communities |
| constructive | Shows what building looks like | the-two-person-studio |
| art | Names the work as an artwork, not a product | keyhole-is-an-installation |
