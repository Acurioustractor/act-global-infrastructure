# Claim: Brilliant islands. No bridges yet.

**The argument:** Each ACT project is a brilliant island. Empathy Ledger doesn't link to JusticeHub case studies. JusticeHub doesn't pull live storyteller voices from the corpus. CivicGraph place pages mention "Empathy Ledger community stories" but the integration is one-way. The world tour talks about "communities already solving it" without naming the 18 orgs and 2,569 voices that prove the claim. **You have not built the bridges that turn islands into a country.** This is the structural critique and the immediate next move.

## What's missing (bridges that need to exist)

| Source | → Target | Currently |
|---|---|---|
| Empathy Ledger storyteller | → JusticeHub case study | Not linked |
| JusticeHub program | → live storyteller voice from corpus | Not pulled |
| CivicGraph place page | → Empathy Ledger storytellers (two-way) | One-way only |
| World tour stop | → corpus orgs at that location | 3 of 4 AU stops empty |
| ACT Global Infra contact | → storyteller in corpus | No linkage |
| Goods on Country bed delivery | → optional storyteller prompt | Not asked |

## What we haven't said yet

- **Never said as a public statement of intent.** This is currently a private criticism.
- Never used to **scope the next two sprints** — the bridges are mostly small code tasks
- Never said in funder pitches as a **transparent admission** — *"Here is what we have built. Here is the part we haven't yet, and here is when it ships."*
- Never paired with the **map view** of the bridges — one diagram, one page, the islands and the missing bridges visible
- Never said: *"The corpus is the country. The projects are the cities. The bridges are what turn them into a thing."*

## Process changes (not new product) needed to build the bridges

- Every Goods bed delivery → optional Empathy Ledger story prompt
- Every JusticeHub program enrolment → story arc at intake / midpoint / exit / 12 months
- Every JCF grant decision → before/during/after stories
- Every CivicGraph place page → two-way storyteller pull
- Every ACT project → designated story moment in lifecycle
