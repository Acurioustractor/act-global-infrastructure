# Claim: The Block architecture maps directly onto JusticeHub. We just got there first.

**The argument:** Jack Dorsey's *From Hierarchy to Intelligence* (Block, 2026) names a pattern: capabilities + world model + intelligence layer + people at the edge. JusticeHub built the same pattern for Australian community-led justice — and the Block essay is a contemporary tech footnote on a structure that was already running. Yunkaporta has been describing it for thirty years and his ancestors for sixty-five thousand. The architecture mapping is the most legible single argument we have for funders who think in platform terms.

## The five-row mapping

| Block layer | JusticeHub equivalent | Status |
|---|---|---|
| **Capabilities** | 1,775 verified community-led models tagged in the Australian Living Map of Alternatives | Built |
| **World model** | Australian Living Map of Alternatives + CivicGraph (100,036 entities, 199,001 relationships, 94.1% match accuracy, $94.6B in funding tracked) | 80% built |
| **Intelligence layer** | Composing capabilities into proactive routing — Hypercerts + Impact-Weighted Quadratic Funding (Allo Protocol) | Designed, not yet wired |
| **Interfaces** | justicehub.com.au, the Australian Living Map of Alternatives Chat, the CONTAINED tour | Live |
| **The edge** | The Aunties, Aboriginal Community Controlled Organisations, tribal councils, Elders, mentors who do the work | Already exists — we just stop interrupting it |

## What we haven't said yet

- **Never deployed publicly.** This entire architecture mapping has only been said inside the synthesis file and the methodology lineage. It has not been on a website, in a deck, or in a public funder pitch.
- Never used as the **opening of an Atlassian Foundation pitch** — Atlassian thinks in this language and would recognise the pattern within thirty seconds
- Never paired with **what each layer cost to build** — the JusticeHub world model has cost <$1M to build over three years; equivalent commercial platforms cost $50M+
- Never said in the **"we just got there first"** form — the Block essay validates an architecture we have been running for two years
- Never used to disqualify **"you should partner with a tech company"** — the platform is built; partnership now means funding the legibility layer, not commissioning a new build
- Never used to argue against **the consultancy that wants to scope a "digital strategy" for the youth justice sector** — the strategy is on the page, the world model exists, the missing piece is funding to wire the intelligence layer
