# How do the 77 project codes connect across financial, storytelling, knowledge, and compliance systems?

> Synthesised from the wiki on 2026-04-13. This article was generated by a query and saved back into Tractorpedia so the next question can build on top of it.

## Answer

## Project Codes as Universal Key

The ACT-XX project code is the single thread that connects every system in the ecosystem. One code (e.g. ACT-HV for The Harvest) appears in bank transactions, Empathy Ledger stories, wiki articles, CivicGraph entities, grant applications, and R&D evidence — all pointing to the same body of work.

### The Seven Financial Buckets

Every bank transaction maps to one of seven spending codes:

| Code | Project | FY26 Spend | R&D |
|------|---------|------------|-----|
| ACT-IN | Infrastructure | \$194K | Core |
| ACT-HV | The Harvest | \$97K | No |
| ACT-PI | PICC | \$66K | No |
| ACT-GD | Goods on Country | \$55K | Supporting |
| ACT-FM | The Farm / BCV | \$55K | No |
| ACT-MY | Mounty Yarns | \$26K | No |
| ACT-JH | JusticeHub | \$1K | Supporting |

### The Full Universe (77 codes)

Beyond financial tracking, 77 project codes cover the entire ecosystem:
- **7 flagship programs** with websites and teams (EL, JH, GD, HV, BV, CS, CORE)
- **14 community projects** (Mounty Yarns, PICC sub-projects, Maningrida, Oonchiumpa, etc.)
- **12 partnerships** (Orange Sky EL, QFCC, AIME, SMART Recovery, etc.)
- **12 creative/events** (Confessional, Storm Stories, festival activations, etc.)
- **32 research, enterprise, and other** projects

### How One Code Crosses All Systems

Take **ACT-UA** (Uncle Allan Palm Island Art):

1. **Financial system**: Bank transactions tagged ACT-UA (via location rule: Palm Island → ACT-HV parent, or manually tagged)
2. **Empathy Ledger**: Storyteller profile, portrait gallery, consent-gated stories, media assets
3. **Wiki**: \`wiki/projects/uncle-allan-palm-island-art.md\` — the durable knowledge article
4. **Narrative store**: Claims about community art, evidence for arts council grants
5. **CivicGraph**: Entity connections to Palm Island community, arts organisations
6. **Grant applications**: Evidence citations pulling from EL media + wiki articles + financial records
7. **R&D evidence**: If art-related development involved novel technology (e.g. EL consent layer), the spend counts

### The Tagging Rules That Connect Them

Three rule layers in \`scripts/tag-statement-lines.mjs\`:
- **62 location rules** in \`location_project_rules\` table: suburb/city → project code
- **26 vendor rules** hardcoded: vendor name → project code
- **48 subscription patterns** in \`subscription_patterns\` table: recurring vendor → project code

Plus travel date overrides that tag spend by where Ben/Nic were physically located on that date.

### Cross-Org Bridging

The \`act_project_code\` field on the Empathy Ledger \`projects\` table bridges across organisations. When a project like PICC Photo Studio (ACT-PS) sits under a partner org, the act_project_code connects it back to the ACT ecosystem. This is how the photo manager shows cross-org media on the ACT organisation page.

### Why This Matters

Without the universal code:
- A bank transaction is just a number
- An EL photo is just an image
- A wiki article is just text
- A grant application starts from scratch every time

With the code, everything is linked. The \$45 Bunnings receipt in Townsville connects to The Harvest, connects to the storytellers, connects to the wiki, connects to the R&D claim, connects to the grant evidence. One code, six systems, one truth.
