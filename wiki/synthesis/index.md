# Synthesis

> Compounding knowledge. When you ask Tractorpedia a question and the answer is non-obvious, the answer is filed here as a permanent article — so the next question builds on top of it.

This folder is the **second-brain compounding loop**. Per the [Karpathy LLM knowledge base pattern](../concepts/llm-knowledge-base.md):

- **Sources** ([[../sources/index|wiki/sources/]]) — one summary per ingested raw file
- **Entities** ([[../people/|people]], [[../projects/|projects]], [[../communities/|communities]]) — the things sources are about
- **Concepts** ([[../concepts/|concepts]]) — frameworks and ideas
- **Synthesis** (here) — the answers to your *questions*: comparisons, contradictions, gaps, "what's the trend across X" — saved so they keep paying off

## How synthesis pages get created

1. You ask `/wiki query "question"`
2. The skill reads relevant articles, synthesizes an answer with citations
3. If the answer is non-trivial, it offers to save it as `synthesis/<slug>.md`
4. The page becomes part of the wiki — the next query builds on top of it
