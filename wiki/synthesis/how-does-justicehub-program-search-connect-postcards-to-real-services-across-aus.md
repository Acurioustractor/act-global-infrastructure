# How does JusticeHub program search connect postcards to real services across Australia?

## Answer

## Postcards to Programs: The JusticeHub Bridge

The QR code on the back of each postcard is the critical link between a felt experience and a practical action. When a judge scans it, they land on JusticeHub — not a generic homepage, but a search filtered to show what actually exists in their jurisdiction.

### What JusticeHub already holds

The data spine is already built:

| Dataset | Scale | Source |
|---------|-------|--------|
| CivicGraph entities | 100,036 | Public sources, daily syncs |
| CivicGraph relationships | 199,001 | Entity resolution |
| Australian Living Map of Alternatives | 1,775 community models | Curated, 128 verified top-tier |
| Justice funding records | 52,000 | GrantScope |
| AusTender contracts | 672,000 | Federal procurement |
| AEC donations | 312,000 | Political donations |

### How the postcard QR should work

Each postcard QR encodes a URL with parameters:

\`\`\`
justicehub.com.au/programs?region=central-australia&needs=diversion,cultural-connection
\`\`\`

The landing page shows:
1. **Map view** — programs in that region, clustered by type
2. **Program cards** — name, who runs it, what it helps with, eligibility, contact
3. **Evidence tags** — Indigenous-led, verified outcomes, ALMA tier rating
4. **Gap indicators** — "No bail support program found in this region" (this is powerful data)

### Program taxonomy (what judges need to find)

Based on the postcard stories, judges will search for:

| Need | Example from postcards | What JusticeHub should surface |
|------|----------------------|-------------------------------|
| **Diversion** | Jackquann — breach, not offence | Community-led diversion programs by region |
| **Cultural connection** | Laquisha — sent 1,500km from family | On-Country programs, cultural camps, Elder-led |
| **Bail support** | Nigel — panicking in court | Bail supervision, bail accommodation, family reconnection |
| **Transport** | "No transport to the nearest program" | Which programs provide transport, which regions have none |
| **Mentoring** | Fred staying present with Xavier | Youth mentoring, case workers, relationship-based programs |

| **Housing** | Multiple stories — no stable home | Youth housing, supported independent living |

### The gap signal

The most powerful feature is not what JusticeHub finds — it is what it doesn't find.

When a judge scans a QR and sees:

> "No culturally-led diversion program found within 200km of this court"

That is a signal. Aggregated across 55 judges across Australia, it becomes a national gap map:
- Which regions have no alternatives to detention?
- Which needs are unmet everywhere?
- Where should the next program be funded?

This data feeds directly into:
- **Grant applications** — "Judges in 12 jurisdictions searched for diversion programs and found nothing"
- **Policy advocacy** — evidence-based gap analysis from judicial demand, not just supply-side mapping
- **GrantScope** — cross-reference gaps against funding opportunities in those regions
- **The Brave Ones Phase 2** — which communities need the experience next?

### The 1,775 community models

The Australian Living Map of Alternatives is the richest layer. Each model has:
- Location and region
- What it does
- Who runs it (Indigenous-led, community-led, government)
- ALMA verification tier (1-5)
- Connected storytellers in Empathy Ledger
- Connected CivicGraph entities

128 models are verified at the top three tiers. These are the programs judges need to know about. The postcard QR puts them one scan away.

### How this compounds

Round 1: 55 judges scan postcards → JusticeHub shows programs + gaps
Round 2: Gap data informs where to build local card decks → more regions covered
Round 3: More judges using JusticeHub → more program data → better search → more referrals
Round 4: Programs that receive referrals through JusticeHub can show outcomes → stronger ALMA tier ratings
Round 5: Funders see demand data + outcome data → fund new programs in gap regions

The postcards are the entry point. JusticeHub is the engine. The compounding happens because every scan teaches the system where Australia is failing its young people — and where the alternatives already exist.
