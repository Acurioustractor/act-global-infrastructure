## How this file is maintained

- **Thursday weekly review:** walk through every cell. Update. Commit the file. Copy to `_weekly-reviews/YYYY-MM-DD.md` as the archived snapshot.
- **When a shipping moment moves:** update the top table immediately.
- **When a claim file is created or retired:** update the deployment matrix.
- **When a person moves from public to warm:** update the campaign's "In relationship" row + `_shared/contacts.md`.
- **When an org makes or withholds a statement:** update the campaign's "Tracking" row + `_shared/orgs.md`.

If you open this file and can't see the whole system on one screen, it's grown too big. Cut. The point is the one-screen view.
