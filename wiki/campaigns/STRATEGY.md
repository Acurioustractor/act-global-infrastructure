# Campaign strategy — operating rhythm, sessions, sequencing

Four campaigns, one system, one rolling calendar. This file is the operating doc — what session runs when, what it produces, how each feeds the others.

Read this first. The campaign briefs (`<campaign>/brief.md`) tell you *what* each campaign is. This file tells you *when and how* the work actually happens.

## The operating logic

Work is organised into **sessions**, not tasks. A session is a 30–120 minute block with one question, one output, and one owner. Sessions chain. Each session's output becomes the input for another.

Sessions live on a **rolling 3-week horizon**:
- **Week 0 (this week):** live sessions, daily cadence, shipping artefacts
- **Week +1:** staging sessions, consolidating the week's outputs, preparing next-week ships
- **Week +2:** anchoring sessions, long-lead scoping so week 0 isn't inventing in real time

Everything beyond +2 stays in the campaign briefs as "queued", not scheduled.

## Ongoing cadence (from week +3 onwards)

**Every Thursday:** weekly review. Copy template, fill five questions, commit. 15 minutes.

**Every Monday:** session planning from the strategy file. 15 minutes. Update this file if the shipping moments have moved.

**Every Friday:** ship day (if a ship is scheduled). No new thinking. Execute the week's work.

**Episode production cadence** (Aesthetics of Asymmetry): one main episode per month, 2 secondary posts between. AutoReason R1 on every main post. AutoReason R2 only if R1's synthesis beats the incumbent (as happened this week).

**CONTAINED cadence:** one leg closes, next scopes, next opens — rolling. Each stop produces at least one claim file and three consented captures.

**Minderoo cadence:** envelope → walk-through → Year 1 decision → Year 2–3 coalition. Each phase has its own sessions when the time comes.

## The cross-campaign doctrine

1. **The stack surfaces the evidence.** CivicGraph + JusticeHub + Empathy Ledger. If a campaign claim can't be traced to the stack or to a named public record, don't ship it.
2. **Named humans in public capacity only.** No "the minister said" without naming. No "a storyteller told us" without consent.
3. **Force-a-trap where there's a claim with a gap.** Not on every campaign — only where a specific public claim has been made and a specific public gap exists.
4. **One campaign's output is another's input.** JoC footage feeds Aesthetics Episode 02. Murri Watch claim feeds Minderoo opening proof. Adelaide scoping feeds Minderoo anchor #10. This is what the weekly review tests.
5. **Silence is evidence.** Track what hasn't been said as rigorously as what has.
6. **Ship > analyse.** A weekly review that concludes "nothing shipped" is a failure signal. Fix it the next week.
