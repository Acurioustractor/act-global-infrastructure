# The Master Book — Chapter Index

> *One book. Twenty chapters. Five parts. Built so the rewriting can compound.*

The master book is the single continuous narrative that holds the entire JusticeHub program — the methodology, the practice, the proof points, the infrastructure, the ask. It is *not* the same as the seven STAY Series method books (which sit at [`series/`](../series/)) and it is *not* the same as the ten community volumes (which sit at [`locations/`](../locations/)). The master book is the one a reader holds to get the whole picture.

## How to use this index

Each chapter has a **status**:

- **`written`** — first full draft exists, ready for revision
- **`drafted`** — partial draft exists, needs completion
- **`sketched`** — argument outlined, not yet written
- **`blank`** — placeholder only

When you sit down to write, pick a chapter that is `sketched` and turn it into `drafted` or `written`. The system pulls from the narrative claim store at `wiki/narrative/justicehub/` so every chapter is anchored in claims that are already verified.
