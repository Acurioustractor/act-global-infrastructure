# ACT-GD core activity: Buyer-supplier matching algorithm (claim under sole trader)

## Hypothesis

We hypothesise that a multi-criteria scoring algorithm will produce buyer-supplier matches with ≥40% first-order conversion vs. ~12% manual baseline.

## Technical uncertainty

Existing fuzzy-matching libraries plateau on Australian organisational data; the dynamic delivery-ceiling profiling problem is unresolved in literature.

## Experiment design

Treatment: algorithm cohort of 60 introductions. Control: 30 manual introductions. Success: ≥40% first-order conversion.

## Outcome

47% first-order conversion (28/60); 78% on-time delivery. Hypothesis supported.
