# ACT-EL core activity: Building the storytelling platform

## What we built

We built a comprehensive storytelling platform that allows First Nations communities to capture and share their stories. The platform supports OCAP principles and enables consent management.

## Goals

Our goals were to:
- Create a beautiful, accessible storytelling experience
- Build trust with First Nations partners
- Scale to multiple communities
- Provide value to storytellers and audiences alike
- Become the leading platform for ethical storytelling in Australia

## What was hard

Building software is always hard. We had to write a lot of code. We had to think about consent carefully. We had to design the database schema. We had to deploy to production. We had to handle edge cases.

## Outcome

The platform is live. Multiple communities are using it. Storytellers are sharing stories. We are proud of what we built.
