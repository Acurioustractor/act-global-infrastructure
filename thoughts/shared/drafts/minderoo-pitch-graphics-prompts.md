# Minderoo pitch — graphic gaps + brand-aligned image-gen prompts

Eight gaps where a single image would carry weight the text can't. Each prompt is
written for an image-gen model and is locked to the existing visual register:
cream stock, hand-bound, letterpress, library/archive feel, no SaaS icons, no
stock-photography lighting, no glossy gradients.

**Brand cheat-sheet (paste at the top of any prompt):**

> Visual register: cream stock paper #faf5ec, lived-in but not distressed, letterpress or hand-painted feel. Typography: Cormorant Garamond italic for headers, IM Fell English or similar for body. Palette: deep purple #4a2560, ochre gold #f0b93a, burnt orange #e8843a, earth grass green #4a6b3a, earth country red #b7410e, charcoal #2b2530. Aesthetic references: Christian Boltanski (memory archives), Edmund Clark (institutional landscape), Theaster Gates (archive-as-practice), Tony Albert (text as object), Daniel Boyd (Indigenous celestial). NOT: slick gradients, corporate dashboard icons, stock photo lighting, glossy 3D, isometric SaaS, neon, AI-art tropes.

## Gap 1 — Map of the Seven (the geography argument)

**Where it lives:** Pitch hero or candidates page header. Currently the geography spread is asserted in text only. A map shows it in a glance.

**Concept:** A hand-drawn, ink-and-wash map of Australia with seven dots. Four dots filled gold (confirmed: Mparntwe NT, Bwgcolman QLD, Minjerribah QLD, Kalkadoon QLD). Three dots open / pencil-outlined for the to-be-confirmed (NSW, VIC/WA, SA/TAS). Place names hand-lettered next to each dot in Cormorant italic. State borders barely visible, drawn like a field-researcher's sketch, not a satellite.

**Prompt:**

> Hand-drawn ink and watercolour map of Australia on cream stock paper, in the style of an ethnographer's field notebook. Seven location dots: four filled in ochre gold (one near Alice Springs labelled "Mparntwe", one off the Townsville coast labelled "Bwgcolman / Palm Island", one near Brisbane labelled "Minjerribah / North Stradbroke", one near Mount Isa labelled "Kalkadoon"). No satellite imagery, no corporate map style, no glossy effects, no compass rose. Square aspect ratio, suitable for printing at A4.

**Variation:** A second version with the dots connected by a faint thread (string-on-pin board feel), to imply relational not transactional.

## Gap 8 — The Heckman / Holding curve

**Where it lives:** Pitch ask section or Living Brain page. The argument is "early intervention beats late". Heckman's curve is the canonical visual; we don't have an ACT-brand rendering.

**Concept:** A hand-drawn return-on-investment curve on graph paper. X-axis: age 0 to 24. Y-axis: return per dollar invested. The curve drops from a high at age 0 to near-zero at age 24. Two zones shaded: green (early intervention, ages 0–6) and red (late intervention, ages 12–24). Annotations in pencil naming what gets funded in each zone. The whole thing on cream graph paper, like a working economist's notebook.

A descending curve drawn in charcoal #2b2530 ink, starting high at age 0 and dropping to near-zero by age 24. ## How to use these

- Run each prompt through whichever image-gen model you trust (Gemini, GPT-Image, Midjourney, etc.)
- Generate 3–4 variants per gap so you can pick by feel
- Drop the best into `apps/command-center/public/minderoo-graphics/<gap-slug>.png`
- Add `<img src="/minderoo-graphics/...">` slots into the relevant page
- The edit-mode `?edit=1` will let you swap them later

If a model returns AI slop (glossy gradients, stock-photo lighting, generic SaaS icons, neon network nodes), reject and re-run. The brand reference is library/archive/hand-made, not platform/dashboard.

## Priority order if you only do three

1. **Gap 1 — Map of the Seven** (highest leverage; the geography question Lucy will ask)
2. **Gap 4 — Cost of Late Intervention folder** (turns a number into evidence)
3. **Gap 5 — Staying** (the methodology rendered emotionally)

Everything else is icing.
