# GrantScope Profile Strategy for ACT

**Date:** 2026-03-19
**Context:** ACT has 1 mega-profile covering 7 projects + 2 project-specific profiles (JusticeHub, PICC). Need to optimize for multi-project grant matching.

## Current State

| Profile | Embedding | Projects | Domains | Saved Grants |
|---------|-----------|----------|---------|-------------|
| **A Curious Tractor** | ✅ | 7 sub-projects | 30 domains | 12 |
| **JusticeHub** | ✅ | 0 | 5 domains | 0 |
| **PICC** | ❌ No mission/domains | 0 | 0 | 0 |

## The Problem

The ACT mega-profile has **30 domains** spanning everything from "indigenous data sovereignty" to "food systems" to "youth justice". This creates a **diffuse embedding** — it matches broadly but not deeply. A grant about regenerative agriculture gets a 58% match instead of 75%+ because the embedding is diluted by justice, digital ethics, and storytelling signals.

**Result:** You're seeing generic matches (Arts QLD historical, ACT Government territory grants) instead of high-confidence targeted opportunities.

## Recommended Strategy: Hub + Spokes

### Keep the ACT mega-profile as the "hub"
- Catches cross-cutting opportunities (e.g. Snow Fellowship, Myer Innovation)
- Shows the full ecosystem story to funders who want to see scale
- Good for: ecosystem-level grants, fellowships for founders, social enterprise accelerators

### Create project-specific "spoke" profiles for each active project

| Profile to Create | Key Domains | Best-Fit Grants |
|-------------------|------------|-----------------|
| **The Harvest + BCV** | regenerative agriculture, agritourism, community hub, cultural tourism, farm-to-table, eco-tourism, creative placemaking, Sunshine Coast Hinterland | Regen ag grants, tourism innovation, community infrastructure, arts council |
| **PICC** (enrich existing) | Indigenous community development, remote service delivery, First Nations enterprise, Palm Island, cultural preservation | NIAA, Indigenous-specific programs, community capacity building |
| **Empathy Ledger** | indigenous data sovereignty, community storytelling, digital ethics, participatory research, consent tech | Tech for good, digital inclusion, research partnerships |
| **Goods on Country** | social enterprise, circular economy, community assets, Indigenous enterprise, laundry services | Social enterprise grants, NDIS-adjacent, community services |

### Why This Works

1. **Sharper embeddings** → Higher match scores (70%+ instead of 58%)
2. **Project-specific pipelines** → Track which grants suit which project
3. **Complementary angles** → Same funder approached from different project angles
4. **Team delegation** → Maddi sees Goods opportunities, Ben sees Harvest

### Implementation Steps

1. **Create Harvest+BCV profile** with targeted mission, domains, geographic focus
2. **Enrich PICC profile** — add mission, domains, geographic focus, regenerate embedding
3. **Run match for each profile** — compare quality vs mega-profile matches
4. **Link saved grants to specific projects** via notes/tags
5. **Optional:** Create Empathy Ledger and Goods profiles if the pattern proves valuable
