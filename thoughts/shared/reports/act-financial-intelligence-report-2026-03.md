### 5.4 Critical Discovery: Phantom Payables ($190K)

**Problem found this session:** Auto-billing integrations (Qantas Business, Uber Business, Virgin, Booking.com, Webflow) create ACCPAY bills with receipt PDFs. But reconciliation creates separate bank transactions instead of matching to these bills.

**Impact:** $190K phantom payables on balance sheet, receipts "missing" from bank transactions even though they exist on the bills.

**Fix:** Un-reconcile and re-match in Xero UI. See `thoughts/shared/plans/xero-bill-matching-fix.md` for detailed match list.

### 5.5 System Limitations (What Can't Be Automated)

| Limitation | Why | Workaround |
|-----------|-----|-----------|
| Bank reconciliation | Xero API doesn't support it | Pre-categorise perfectly, rely on JAX |
| Bank rules | Not in Xero API | Use vendor_project_rules in Supabase |
| BAS lodgement | UI only | Generate worksheet, accountant reviews |
| Physical receipts | No digital source | Xero mobile app for photo capture |
| Qantas hotel/agent bookings | Not shared by Qantas Business | Manual download |
| Account code assignment | Need chart of accounts rules | Build vendor → account code mapping |

### 5.6 Recommendations for Perfect System

1. **Fix bill matching** — un-reconcile and re-match (removes $130K phantom payables)
2. **Add account code rules** — map top 50 vendors to default Xero account codes
3. **Build business division tagging** — map project codes to Xero's Business Divisions tracking category
4. **Set up Xero webhooks** — real-time sync instead of 6-hour polling
5. **Add vendor ABN enrichment** — lookup from ABR, critical for GST validation
6. **Build payables reconciliation** — cross-reference AUTHORISED bills against bank transactions weekly
7. **Monthly financial intelligence report** — auto-generate this report every month
8. **R&D documentation pack** — link git commits + calendar events + Xero transactions per project
